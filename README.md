# PHP Newsletter

**Version:** 7.1.*

PHP Newsletter is a web-based email marketing system for managing and sending bulk email campaigns directly from your website.

---

## 🚀 Features

### Email Sending
- SMTP, mail() and sendmail support
- Bulk email sending
- Attachments support
- HTML and plain text formats
- Custom email headers (Precedence, List-Unsubscribe)

### Templates & Personalization
- WYSIWYG editor
- Dynamic data (macros)
- Email personalization
- Multi-encoding support

### Subscribers Management
- Subscriber database management
- Categories support
- Manual and automatic subscription
- Unsubscribe functionality

### Import / Export
- Import: TXT, CSV, Excel (XLS, XLSX), ODS
- Export: TXT, Excel (XLSX)

### Statistics
- Sent / failed emails tracking
- Email open tracking (HTML only)
- Link click tracking
- Export reports to Excel

### Automation
- Cron-based scheduling
- Background processing
- Queue system

### Security
- DKIM support
- Flexible SMTP configuration

---

## ⚙️ Requirements

- PHP >= 8.2
- MySQL >= 5.6
- Apache (mod_rewrite) or Nginx

### PHP Extensions:
- mbstring
- zip
- curl
- iconv
- gd
